## 개요

*Bungie*의 1인칭 슈팅 온라인 게임 *Destiny 2*에서는 일부 상인들이 특정한 주기마다 임의의 아이템을 판매한다. 이 웹 사이트는 *Bungie.net Platform API*를 사용하여 판매 아이템 정보를 가져와서 데이터베이스에 기록, 조회하는 기능을 제공하며, 나아가 인공지능 기반의 구매 추천 기능 또한 제공한다.
